현장 탐구
Physical Grounding
데이터가 없는 곳에 진짜 인사이트가 있다
  • 미묘한 표정, 망설임, 소음 — 에이전트가 절대 학습하지 못하는 것들
  • 비정형적 맥락을 포착하는 유일한 존재
지적 심화
Tech Deep-Dive
툴을 쓰는 사람과 기술을 설계하는 사람은 다르다
  • 논문과 특허 속 기술적 특이점을 서비스 인터랙션으로 번역
  • 에이전트에게 시킬 일의 한계를 아는 아키텍트
교차 지능
Cross-Sync
혼자 에이전트와 대화하면 편향만 강해진다
  • 개발자, 데이터 사이언티스트, 인문학자와의 지적 마찰
  • 입체적 관점이 대체 불가능한 리더십을 만든다
자산화
Knowledge Assetization
협업의 과정 자체가 포트폴리오다
  • 실패한 프롬프트, 성공한 로직, API 규격 — 나만의 AX 플레이북
  • 시안 한 장보다 방법론 하나가 오래간다
Conclusion

디자이너 R&R — 기기 간 경계 없는 지능의 Handoff 설계
①
컨텍스트 연속성 설계
기기가 바뀌어도 에이전트가 대화·상태·진행 중인 태스크를 그대로 이어받을 수 있도록 정보의 흐름과 인계 시점을 정의
②
주도권 전환 신호 설계
어떤 상황에서 어떤 기기가 주도권을 가져야 하는지 판단 기준과 전환 순간을 사용자가 자연스럽게 인지할 수 있는 시각·청각·촉각 피드백 설계
③
멀티 디바이스 UI 문법 정의
폰·워치·가전 등 각기 다른 화면 크기와 인터페이스 조건에서 일관된 경험을 제공할 수 있도록 기기별 표현 규칙과 공통 언어 정립
