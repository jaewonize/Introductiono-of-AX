## CSS 클래스 구조

### 슬라이드 타입
- `.s-full` — 풀스크린 (표지/파트표지/간지). 배경 그라데이션 클래스 별도 적용
- `.s-page` — 일반 콘텐츠 페이지. 배경 `#0a0a0a`

### 풀스크린 구성요소
- `.plabel` — 소제목 (파트명, 챕터명)
- `.ptitle` — 본제목
- `.psub` — 부제

### 콘텐츠 페이지 구성요소
- `.plabel` — 페이지 번호 레이블 (예: "1.1" — Part 표기 제거됨, 네비게이션 바와 중복)
- `.ptitle` — 페이지 제목
- `.pcap` — 도입 카피
- `.body` — 콘텐츠 영역 (flex)
- `.body.col` — 세로 방향 콘텐츠 영역

### 그리드 헬퍼
- `.g2` `.g3` `.g4` — 2/3/4컬럼 그리드
- `.fw` — width 100%
- `.wf` — 와이어프레임 플레이스홀더
- `.wf.circle` — 원형 와이어프레임

### 그라데이션 배경
- `.g-cover` — 커버
- `.g-p1` ~ `.g-p5` — 파트별
- `.g-chap` — 챕터 간지
