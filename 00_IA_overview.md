# 00. AI Agent Experience — IA Overview

## 기술 명세

### 캔버스
- 고정 해상도: 1920×1080 (16:9)
- 브라우저 뷰포트에 맞게 자동 스케일 (Math.min 방식)
- 여백은 배경 컬러로 처리 (레터박스 방식)
- 리사이즈 시 자동 재계산

### 네비게이션
- 상단 고정 바. 풀스크린 이미지 위에 흰색 글씨 오버레이
- 왼쪽: AI Agent Experience (타이틀)
- 오른쪽: Part 1~5 탭
- 모든 탭 호버 시 해당 파트 페이지 목록 드롭다운
- Part 4만 챕터명 그룹 헤더(클릭 불가, 제목 표시) + 하위 페이지 목록

### 슬라이드
- 표지/파트 표지/챕터 간지: 블랙 배경 + 추상적 컬러 그라데이션 (CSS로 구현)
- 일반 콘텐츠 페이지: 스크롤 없이 1920×1080에 맞게
- 전환 효과: Reveal.js, 드러내기(Reveal) 방식, 좌→우 방향

### 폰트
- Noto Sans KR (Google Fonts)
- 한글/영문 모두 적용

### 라이브러리
- Reveal.js (슬라이드 전환, 키보드 네비게이션, 풀스크린)

### Export
- PPTX 16:9, 1920×1080
